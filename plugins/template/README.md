# Template Plugin

This is a template plugin used for testing the CI infrastructure for FiftyOne Labs plugins.

## Purpose

This plugin serves as a reference implementation and testing framework for:
- FO version compatibility testing
- Plugin registration and operator instantiation
- Basic FiftyOne API usage

## Testing

This plugin includes a comprehensive test suite that validates:
- Plugin compatibility with the latest FiftyOne version
- Proper plugin registration
- Operator functionality

See `tests/` directory for test implementations.
